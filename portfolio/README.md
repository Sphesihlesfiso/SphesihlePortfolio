# Personal Portfolio

A personal portfolio website built with **React.js** to showcase projects, skills, and experience. The site includes a contact form powered by **EmailJS**, allowing visitors to send messages directly without a backend.

---

## Features

- Project and skills showcase
- Responsive and modern UI
- Contact form with EmailJS integration
- Fast and lightweight React-based setup

---

## Tech Stack

- React.js
- EmailJS
- HTML
- CSS
- JavaScript

---

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/yourusername/personal-portfolio.git
cd personal-portfolio
npm install
npm start
```
